# IBM Data Science Certification Projects

This repository contains a curated collection of hands-on projects, assignments, and Jupyter notebooks completed as part of the IBM Data Science Professional Certificate. Each project demonstrates real-world applications of essential data science skills and tools.

## Key Technologies & Libraries

- **Python**: Primary programming language for all projects
- **Pandas**: Data analysis and manipulation
- **NumPy**: Numerical computations and efficient array operations
- **SQL**: Querying and managing relational databases
- **Scikit-Learn**: Machine learning algorithms and workflows
- **Matplotlib & Seaborn**: Data visualization and exploratory analysis
- **BeautifulSoup4**: Web scraping and data extraction from HTML
